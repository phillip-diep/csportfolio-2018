Linking a CSS page:

In order to stylize my HTML page, I had to first link a CSS page to it. I had to use the <link> tag to link the css code.
The code would look like this: < link rel="stylesheet" href="index.css" type="text/css" />
As you can see, the link tag is used: Href is used to give the link of the CSS sheet.

Background:

To make the background, I first went to my CSS code. Using "body" I made the background. First, I have to link the image I wanted to use using "background-image." In this command, I used url() to link the url of the image that I wanted.

This is the image I wanted to use.

I then had to make sure that the image covered the whole webpage. I used "background-size:cover" to do this. I also had to make sure that the image would stay in place as the user scrolled down. I used "background-attachment:fixed" to do this. In the end, the code looked like this: body{

background-size: cover;
background-attachment: fixed; }

I did this by using the <div> tag and “class”. I then took that class, and put it in my CSS file: I edited the height and width of the columns here. The file looked like this:

width: 50%;
Height: auto;
Next, I made a <div> + class tag that contained the two column <div>s. In the CSS file, I used “display” and set it to “flex” so that the two columns were now in 1 flex block. The file looked like this:
.wihl{
Display: flex;
…}

Animations:

To make the pictures move, I used an animation. I first made the animation using @keyframes. I set the animation to make the margin bigger on the left and right sides, which would made the pictures move away from each other. The animation looked like this:
@keyframes mymove{
50% {margin: 0px 100px 0px 100px;} }
After this, I then used "animation" inside the .imges class (the class that contained the whole image block) to play the animation for 30 seconds infinitely. It looked like this: .imges{
animation:mymove 30s infinite; }
