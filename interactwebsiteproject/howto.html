How To:

Interactive Website

What did I create?

I created a website that uses user input to determine the best times to cook each side of a steak for. The inputs are the thickness of the steak and the level of cooking they wanted.

How does the project work overall?

The project works by using the DOM to manipulate how the website appeared. Elements would be stored in JavaScript using .getElementById. They would then have their properties changed in many ways: css styling could be changed using .style, html itself could be changed using .innerHTML. The code also used .addEventListener to many buttons. This command would run functions when a certain event happened to the assigned object. I also used a if / else if statement to create input/output system.

What are some specifics of the code and how do those work?

In the code, .getElementById was used heavily. An example of it is the line: 

 
    var input = document.getElementById("input");

This line of code works by first creating a variable in JS, and then assinging it to the .getElementById. The .getElementById is writen always as document.getElementById("name"). It looks for anything with the same id as what is in the quotation marks and calls it to the JS code.

In the code, .style was used to change the styles of many elements. An example of it is in the lines: 

 
    opening.style.opacity = "0";
    input.style.display = "flex";
    error.style.display = "block";

In these lines of code, there are many .style codes being used. In the first line, .style is paired with .opacity because that is what we are changing. When using .style, the specific css property being changed follows as .NAME. The first line will change the opacity of "opening" to 0, change the display of "input" to flex, and change the display of "error" to block. These .style changes, with other ones, are how I got my different slides to tranistion from one to the other smoothly.

In the code, .innerHTML was used to change to html of the results. An example of it is in the lines: 

 
    side1.innerHTML = “<h4> 5 minutes </h4>”
    side2.innerHTML = “<h4> 3 minutes </h4>”

These 2 pieces of code are one result for the output. There are 2 innerHTMLs being changed: side1 and side2. There two variables are div elements, so when the innerHTML is changed, the results will be displayed in the divs in a <h4> style.

In the website, there are many buttons that have many functions. To have them cause events to happen, .addEventListener has to be used. An example of it is in the lines: 

 
    exitAlert.addEventListener("click", function() {
    error.style.top = "-37vw";
    });

This piece of code is a complete .addEventListener. There is a variable which the eventListener will be added to (exit.alert), a event that is being listened for ("click"), and a fucntion that is run when the event happens.

In the website, the output is determined using a system of if / else if statements. There are 4 different if / else if statements: each one starts by looking for what cooking was choosen. After the cooking has been found, the thickness value is looked at to find what range it fits in, and then the output value for that range is displayed.
